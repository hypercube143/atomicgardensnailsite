API Documentation

/cotd/ all things Creature of the Day

/cotd/upload [POST] upload a Creature of the Day
---- JSON
-------- "title" [str]
-------- "description" [str]
-------- "date" [optional] [str] [yyyy-mm-dd] - if not specified, will default to today

/cotd/edit [POST] edit fields for a Creature of the Day
---- JSON
-------- "title" [optional] [str]
-------- "description" [optional] [str]
-------- "date" [optional] [str] [yyyy-mm-dd] - if not specified, will default to today
-------- "entry_no" [optional] [int] NOTE: at least one field must be specified

/cotd/delete [DELETE] delete a Creature of the Day
---- JSON
-------- "date" [optional] [str] [yyyy-mm-dd] - if not specified, will default to today

/cotd/get [GET] get a single Creature of the Day from a date
---- JSON
-------- "date" [str] [yyyy-mm-dd]

/cotd/get_today [GET] get today's Creature of the Day
